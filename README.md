# hyprprop
xprop for Hyprland


## Dependencies
- `hyprland` obviously
- [hyprevents](https://github.com/vilari-mickopf/hyprevents) capture window changed events to rerun slurp
- `slurp` to select an area on the screen

## Install
```bash
yay -S hyprprop-git
```
or
```bash
sudo make install
```

## Usage
Just run
```bash
hyprprop
```
and select desired window
